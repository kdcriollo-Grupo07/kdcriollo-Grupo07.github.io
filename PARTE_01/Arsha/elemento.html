Encontrar elemento HTML

La forma más fácil de encontrar un elemento HTML en el DOM es utilizando la identificación del elemento.

Se puede realizar la búsqueda mediante:

  • Id (Identificación)
  • TagName (Nombre de etiqueta)
  • ClassName (Nombre de la clase)
  • QuerySelector (Selector Css)
  • Por colecciones de objetos HTML
