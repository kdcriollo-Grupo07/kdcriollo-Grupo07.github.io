Cambio de HTML

Cambio de contenido HTML

La forma más sencilla de modificar el contenido de un elemento HTML es mediante la innerHTML propiedad.

Métodos que podemos utilizar

  • Cambia el contenido de un html: document.getElementById(id).innerHTML = new HTML, en el primer ejemplo se observa el cambio de contenido de un <"p">
  • Cambia el valor de un atributo: document.getElementById(id).attribute = new value, el segundo ejemplo cambia el valor del atributo src de un <"img"> elemento
  • Contenido HTML dinámico: JavaScript puede crear contenido HTML dinámico, en el tercer ejemplo se observa la hora de Ecuador Fecha: jue 08 dic 2022 08:42:37 GMT-0500 (hora de Ecuador)
  • documento.escribir(): En JavaScript, document.write() se puede usar para escribir directamente en el flujo de salida HTML
    Nota: Nunca lo use document.write() después de cargar el documento. Sobrescribirá el documento.
