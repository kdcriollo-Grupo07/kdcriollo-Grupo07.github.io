HTML DOM Navegación

HTML DOM NAVEGACIÓN

Con HTML DOM, puede navegar por el árbol de nodos utilizando relaciones de nodos.

Métodos que podemos utilizar

  • Relaciones de nodo: Los nodos del árbol de nodos tienen una relación jerárquica entre sí.
    Los términos padre, hijo y hermano se utilizan para describir las relaciones.
    1. En un árbol de nodos, el nodo superior se llama raíz (o nodo raíz)
    2. Cada nodo tiene exactamente un padre, excepto la raíz (que no tiene padre)
    3. Un nodo puede tener varios hijos.
    4. Los hermanos (hermanos o hermanas) son nodos con el mismo padre
  • Navegación entre nodos: Puede usar las siguientes propiedades de nodo para navegar entre nodos con JavaScript:
    1. parentNode
    2. childNodes[nodenumber]
    3. firstChild
    4. lastChild
    5. nextSibling
    6. previousSibling
  • Nodos secundarios y valores de nodoUn error común en el procesamiento de DOM es esperar que un nodo de elemento contenga texto.
    1. El nodo del elemento <"title"> (en el ejemplo) no contiene texto.
      Contiene un nodo de texto con el valor "Tutorial DOM".
      Se puede acceder al valor del nodo de texto mediante la innerHTML propiedad del nodo
  • Nodos raíz DOM: Hay dos propiedades especiales que permiten el acceso al documento completo:
    1. document.body- El cuerpo del documento.
    2. document.documentElement- El documento completo
  • Propiedad de nodeName : La nodeName propiedad especifica el nombre de un nodo.
    1. nodeName es de solo lectura
    2. nodeName de un nodo de elemento es el mismo que el nombre de la etiqueta
    3. nodeName de un nodo de atributo es el nombre del atributo
    4. nodeName de un nodo de texto siempre es #text
    5. nodeName del nodo del documento siempre es #document
  • Propiedad nodeValue: La nodeValuepropiedad especifica el valor de un nodo.
    1. nodeValue para nodos de elementos es null
    2. nodeValue para nodos de texto es el texto mismo
    3. nodeValue para los nodos de atributo es el valor del atributo
  • Propiedad de tipo de nodo: La nodeType propiedad es de solo lectura. Devuelve el tipo de un nodo.
