Elementos HTML DOM (nodos)

Creación de nuevos elementos HTML (nodos)

Para agregar un nuevo elemento al HTML DOM, primero debe crear el elemento (nodo de elemento) y luego agregarlo a un elemento existente.

Métodos que podemos utilizar

  • insertBefore(): El appendChild() método del ejemplo anterior, agregó el nuevo elemento como el último hijo del padre.
    Si no quieres eso, puedes usar el insertBefore()
  • Eliminación de elementos HTML existentes: Para eliminar un elemento HTML, utilice el remove()
    El remove() método no funciona en navegadores más antiguos
  • Eliminación de un nodo secundario: Para los navegadores que no admiten el remove() método, debe encontrar el nodo principal para eliminar un elemento
    Hay una solución común: busque el elemento secundario que desea eliminar y use su parentNode propiedad para encontrar el elemento principal
  • Sustitución de elementos HTML: Para reemplazar un elemento en HTML DOM, use el replaceChild()
